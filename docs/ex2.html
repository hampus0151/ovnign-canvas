Animation

När vi animerar med canvas så ritar vi om bilden steg för steg. Vi behöver rensa och sedan rita.

Din uppgift är att:

  1. Testa vad som händer utan clearRect / tömma canvas.
  2. Byt färg på boxen.
  3. Skapa ytterligare en box/cirkel eller liknande och få den att studsa mot box.y HEIGHT
  4. Lägg till åtminstone en till parameter på boxen när vi skapar den, som färg, storlek.
